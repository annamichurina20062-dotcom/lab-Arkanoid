# Arkanoid

Игра на C++ с использованием библиотеки SFML.

## Платформа

Windows (MSYS2 / UCRT64)

## Зависимости

- [SFML 2.5+](https://www.sfml-dev.org/download.php) — графика, окно, система

Установка через MSYS2:
```bash
pacman -S mingw-w64-ucrt-x86_64-sfml
```

## Сборка и запуск

```bash
g++ src/*.cpp -Iinclude -lsfml-graphics -lsfml-window -lsfml-system -o arkanoid.exe
./arkanoid.exe
```

## Управление

| Клавиша | Действие |
|--------|----------|
| ← → | Движение ракетки |
| Space | Отпустить шарик (после прилипания) |
| P | Пауза |
| R | Рестарт после поражения |
| Esc | Выход |

## Механики

- Блоки с разным количеством HP (от 1 до 4)
- Неразрушаемые блоки
- Блоки-ускорители
- Бонусы: расширение/сужение ракетки, изменение скорости, липкий шарик, дополнительный шарик
